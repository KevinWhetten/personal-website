Tracker for The Legend of Zelda: Ocarina of Time Randomizer

The Legend of Zelda: Ocarina of Time is one of my favorite games ever. I have a ton of nostalgia for it. The only problem is that I know the game so well, it's not as fun on replay. However, with the Legend of Zelda: Ocarina of Time Randomizer (OoTR for short), new challenges are presented every time you play, making the game new again. It takes all the items available in the game and randomizes their locations. I enjoy playing it so much that I created my own personal tracker. It contains a list of places where items can be found, an inventory screen for tracking items obtained, and a hint section where in-game hints can be recorded quickly. It is available for anyone to use at www.ootr-tracker.com. It is currently incomplete, but I am actively working on it. The github repository can be found here.
